Modern Chat - Полноценный мессенджер с реальным временем
Современное веб-приложение для общения в реальном времени с использованием FastAPI (бэкенд) и React (фронтенд).

✨ Возможности
Основные
🔐 Аутентификация и авторизация (JWT токены)

💬 Чат в реальном времени (WebSocket)

👤 Профиль пользователя с настройками

💎 Тарифы и подписки (Stripe интеграция)

📱 Адаптивный дизайн (работает на всех устройствах)

🎨 Современный UI с анимациями

Безопасность
🔒 Хеширование паролей (bcrypt/sha256)

🛡️ Защита от DDoS (лимиты запросов)

🍪 Secure cookies для хранения токенов

🚫 XSS и CSRF защита

🛠 Технологии
Бэкенд
FastAPI - веб-фреймворк

SQLAlchemy - ORM для работы с БД

AuthX - JWT аутентификация

WebSocket - реальное время

Asyncpg/PostgreSQL - база данных

Stripe - платежная система

Фронтенд
React 18 - UI библиотека

TypeScript - типизация

Vite - сборщик

Axios - HTTP клиент

React Router DOM - навигация
